Widget Capabilities

Zero Dependencies

A ~45kb gzipped JavaScript bundle that executes effortlessly. No bulky React trees injected onto your hosting payload.

Fully Themed

Pass your CSS hex codes and font families directly into the configuration to map the UI to your brand aesthetics.

Shadow DOM Isolation

Every element is shielded via a Shadow DOM context so none of your existing CSS frameworks (like Tailwind) will bleed into or break the chat interface structure.
